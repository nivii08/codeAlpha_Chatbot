# CodeAlpha Chatbot Project

## Description
This is a basic rule-based chatbot developed using Python. It responds to user inputs using predefined conditions and demonstrates the use of functions, loops, and dictionaries.

## Features
- Handles multiple user inputs
- Provides predefined responses
- Includes a custom response for "internship"
- Clean and structured code

## How to Run
1. Open terminal
2. Run: python chatbot.py
